## Key conventions (non-obvious)

- **AI is Claude.** All AI runs in Supabase Edge Functions calling Claude Sonnet
  4.6 (`claude-sonnet-4-6`) via `supabase/functions/_shared/anthropic.ts`
  (`callClaudeJSON`). Auth-gated; secret `ANTHROPIC_API_KEY`. No JSON mode and no
  assistant-prefill — instruct JSON in the system prompt; give generous
  `max_tokens` (Sonnet truncates into invalid JSON if too low).
- **Crawler-facing SEO** (title/meta/JSON-LD) is generated by
  `scripts/prerender.mjs`, NOT React `<Helmet>`/`SEOHead`. Change both when
  editing a page's meta.
- **Canvas tools** share `src/hooks/canvas/` and `src/utils/canvas/canvasExporter.ts`.
  Tools persist into Projects as artifacts (`project_artifacts`, dispatched by
  `artifact_type` in `src/pages/ArtifactViewer.tsx`).
- **Deploy edge functions:** `npx supabase functions deploy <name> --project-ref wqaplkypnetifpqrungv`.
